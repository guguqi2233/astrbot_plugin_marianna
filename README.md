# 玛丽亚·特蕾莎·冯·哈布斯堡 — AstrBot 人格插件

[![AstrBot](https://img.shields.io/badge/AstrBot-4.11+-blue)](https://github.com/Soulter/AstrBot)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

哈布斯堡贵族少女人格插件 —— 傲娇 + 诱导性病娇，集成对话总结、用户画像学习、长期记忆与动态数值调整。

## ✨ 功能特点

- 🎭 **动态人格状态机**：7 种状态（冷傲贵族 → 傲娇试探 → 甜蜜诱导 → 潜伏之藤 → 锁定·命定之人 → 焦虑·崩溃边缘 → 优雅崩坏），根据互动自动切换
- 📊 **数值追踪系统**：好感度、病娇值、锁定进度、优雅值等 7 项核心属性实时变化
- 🧠 **长期记忆**：集成 Mnemosyne 插件，自动存储并检索重要对话
- 📝 **对话总结**：自动/手动生成对话摘要，提炼关键信息
- 👤 **用户画像学习**：通过 LLM 分析对话，动态构建用户兴趣、喜好、性格画像
- ⚙️ **WebUI 可视化配置**：无需修改代码，在管理面板调整所有参数
- 💬 **丰富指令**：6 个交互指令，随时查看状态、画像、总结等
